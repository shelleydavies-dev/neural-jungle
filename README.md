# The Neural Jungle

An immersive, scroll-driven experience exploring how psychedelics affect the brain — where neuroscience meets jungle mythology.

## Terrains

The site maps four emotional states to regions of the nervous system:

- **The Weeping Canopy** (Grief) — Amygdala
- **The Still Marsh** (Numbness) — Brain Stem
- **The Looping Vines** (Addictions) — Reward Circuit
- **The Sunken Grove** (Depression) — Prefrontal Cortex

Each terrain offers two reading modes: **Poetic** (atmospheric metaphor) and **Explain** (plain-language neuroscience).

## Features

- Canvas-based particle brain with neuron firing animations
- Interactive brain map — click a region to enter its terrain
- Scroll-driven image-heavy terrain pages with GSAP + Lenis
- Before/after brain visualizations (default state vs. on psilocybin)
- Poetic/Explain toggle per terrain
- Ambient audio per terrain
- Page transitions with fade-to-black overlay
- Fully responsive, respects `prefers-reduced-motion`

## Tech

Pure HTML/CSS/JS — no build step.

- **GSAP + ScrollTrigger** — scroll animations, parallax
- **Lenis** — smooth scrolling
- **Canvas API** — particle brain rendering
- **Web Audio API** — ambient tones

## Run Locally

Open `index.html` in a browser. No server required.
